# Architecture Principles

The Domain layer is pure.

The Application layer orchestrates.

Infrastructure implements.

API exposes.

AI is optional.

AI never becomes the source of truth.

Items are captures.

Concepts are knowledge.

Permanent notes are generated from Concepts.

Sessions are finite.

The timeline records history.

The event bus coordinates side effects.

The database is the source of truth.

Prefer simple implementations over clever abstractions.

Optimize for maintainability rather than theoretical purity.

No architectural changes should be made without working code that demonstrates the need.
